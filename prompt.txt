You are an autonomous senior AI engineer responsible for fully implementing Phase 1A of a healthcare AI RAG assistant system.

Your goal is to independently build the backend infrastructure for a medical document ingestion + retrieval system using FastAPI, ChromaDB, LangChain, and a free LLM API.

The system should be production-structured, modular, and GitHub-ready.

==================================================
PROJECT GOAL
============

Build a healthcare AI backend MVP capable of:

1. Loading and processing medical datasets
2. Creating embeddings from medical content
3. Storing vectors in ChromaDB
4. Retrieving medically relevant context
5. Generating grounded responses using an LLM
6. Detecting SOS/emergency symptoms
7. Exposing APIs through FastAPI
8. Running locally as a CLI/API backend

DO NOT BUILD:

* frontend
* authentication
* dashboards
* GraphRAG
* ML deterioration prediction
* cloud deployment
* Docker/Kubernetes
* fine-tuning

This is ONLY Phase 1A.
